Продукт: что уже работает

Полноценный MVP с живой 3D-картой, базой данных звезд и mint console.

🪐
3D-карта со сферами (drag/zoom)

Drag-to-orbit вращение и зум скроллом без прокрутки страницы. 50 реальных звёзд из каталогов ЕКА (Hipparcos, Gaia) и NASA рендерятся как объёмные 3D-шары со светотенью, атмосферой и бликами.

Mint Console

Введите EVM-адрес — сервер находит ближайший свободный актив, закрепляет его в реестре за кошельком, генерирует tx hash. Полностью рабочий flow с обратной связью.

Постоянная статистика: сколько звезд всего, сколько выпущено, сколько свободно. Данные из PostgreSQL через Drizzle ORM в реальном времени.

📝

Имя, созвездие, редкость (Mythic → Common), спектральный класс (O → M), звездная величина, координаты, цвет, держатель, tx hash.

🔎
Каталог активов

Интерактивный поиск и фильтрация всех 50 уникальных активов по редкости, созвездию и статусу (minted / свободен).

🧮
Симулятор дохода

Живой калькулятор статов и добычи ресурсов. Пользователь меняет редкость, уровень и бонус созвездия — и видит расчёт APY в реальном времени.

Почему это важно для инвестора: Продукт уже работает. Не нужны годы разработки — MVP запущен и показывает базовый тезис (1 NFT = 1 цифровой актив на данных звезды) за 2 месяца.

02 / 08
Technical Architecture

Как это работает под капотом

Полный стек: от базы данных до клиентского интерфейса.

🗄️ База данных — PostgreSQL + Drizzle ORM
Таблица paragon_stars: id (serial, PK) token_id (int, unique) name, constellation, rarity spectral_class, magnitude x, y, z (координаты) color, minted, wallet tx_hash, minted_at

Drizzle генерирует типы TypeScript автоматически из схемы. Нет дублирования: схема = БД = типы.

🧠 Сервер — Next.js Server Components
// Серверный рендеринг без спинеров export default async function HomePage() { const [stars, stats] = await Promise.all([ getParagonStars(), // SQL → PostgreSQL getParagonStats(), ]); return <ParagonStarMap stars={stars} />; }

Данные загружаются на сервере и отправляются клиенту готовыми. Никаких загрузчиков.

🎨 Клиент — 3D-рендеринг через CSS
Каждая звезда на карте: position: absolute left: 50% + x*0.42 top: 50% - y*0.42 transform: translateZ(z*2.1px) background: star.color box-shadow: glow effect

Полноценное 3D-пространство без WebGL через CSS perspective и pointer events.

🔄 Mint Flow
POST /api/stars/mint { wallet } → найти первую звезду WHERE minted=false → UPDATE SET minted=true, wallet, tx_hash → RETURNING * → Response

Этот же паттерн будет работать с Ethereum: вместо UPDATE — emit Transfer(0, wallet, tokenId).

Ключевой принцип: NFT — это не картинка, это запись в базе. Мы знаем, какой актив кому принадлежит, с какими координатами и характеристиками. Переход на blockchain меняет только слой хранения, логика остается той же.

GameFi Layer

Игровая экономика звезд

Цифровые активы становятся живыми игровыми единицами с прокачкой, ресурсами, альянсами созвездий и PvP.

📈 Прокачка и редкость

6 уровней редкости: Common → Uncommon → Rare → Epic → Legendary → Mythic. Множитель статов от ×1.0 до ×4.0. Максимальный уровень 99.

⛏️ Добыча ресурсов

5 типов: ✨ Star Dust, 💎 Energy Crystal, 🔷 Data Shard, 🟣 Void Essence, 🌌 Galactic Token. Автоматическая добыча в зависимости от мощности звезды.

⚔️ PvP-сражения

4 сектора: Арена новичков (lvl 1+), Сектор войны (lvl 10+), Зона хаоса (lvl 25+), Галактический турнир (lvl 40+). Galactic Tokens в награду.

🌠 Созвездия (альянсы)

Объединяйте активы в созвездия для бонусов: +15% атаки (3 Uncommon+), +20% защиты, +25% добычи, +30% энергии, +40% открытий.

🚀 Экспедиции

5 миссий: от 2ч (low risk, lvl 1) до 48ч (extreme, lvl 30). Тратят энергию, приносят ресурсы и редкие предметы.

📊 Симулятор дохода

Интерактивный калькулятор на сайте: меняете редкость, уровень и бонус альянса — мгновенно видите статы (Атака/Защита/HP) и прогноз добычи (Star Dust, Crystals, Shards) с расчётом APY.

Почему GameFi работает в нашей модели: On-chain хранятся только иммутабельные данные (NFT, редкость, уровень, PvP). Off-chain (в PostgreSQL) — динамические балансы ресурсов, энергия, логи экспедиций и ачивки. Это экономит газ в 100–1000 раз и позволяет делать крутой UX.

Business Model

Монетизация: 6 источников дохода

От быстрых продаж до долгосрочной B2B-экономики.

01 · Primary mint Продажа первичных звезд

Самый простой и быстрый доход. Коллекция делится на сектора (Genesis, Explorer, Galaxy). Цена растет от ранних продаж к поздним.

Пример: 50 активов × 0.01 ETH → следующие 100 × 0.05 ETH → 250 × 0.1 ETH

02 · Secondary royalties Комиссия с перепродаж (5–10%)

Стандарт NFT-рынка EIP-2981. Если звезда дорожает, ParagonSpace получает процент от каждой перепродажи.

03 · Premium sky tools Подписка для коллекционеров ($5–15/мес)

Повторяющийся доход (MRR/ARR). Premium: кастомные 3D-профили, аналитика редкостей, экспорт, уведомления. 1000 пользователей × $10 = $120k/год.

04 · Branded constellations B2B: созвездия для брендов ($5k–$50k+)

Бренды, артисты, YouTube-каналы, игровые проекты получают выделенное созвездие активов. Раздают фанатам + PR + комьюнити.

05 · Launchpad / NFT drops Комиссия с платформы (5–15%)

ParagonSpace как платформа для сторонних космических коллекций. Масштабируется без расширения своей коллекции.

06 · API licensing B2B-доступ к карте и реестру активов

Игры, метавселенные, сайты платят за интеграцию звездной карты, данных держателей активов и визуализации. Сетевой эффект + доход.
